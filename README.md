# Cookies to TXT 🍪

[Español](#español) | [English](#english)

---

## English

**Cookies to TXT** is an open-source fork optimized and designed to work flawlessly on both desktop browsers and Firefox for Android. This extension allows you to capture and download your cookies in standard Netscape format directly to a text file in a fully private and local manner.

### 🚀 Features of this Fork:
* **True Hybrid Design:** Responsive UI that adapts seamlessly to desktop monitors and mobile touchscreens.
* **Android Fix:** Resolves the critical download-blocking bugs present within the Firefox mobile environment.
* **Native Dark Mode:** Color palette fully optimized for low-light environments.
* **Privacy:** 100% local processing. Your information never leaves your device.

### 🛠️ Development Installation:
1. Clone this repository.
2. Open Firefox and navigate to `about:debugging`.
3. Click on "This Firefox" and select "Load Temporary Add-on".
4. Choose the `manifest.json` file from the project folder.

## Español

**Cookies to TXT** es un fork de código abierto optimizado y diseñado para funcionar de manera impecable tanto en navegadores de escritorio como en Firefox para Android. Esta extensión te permite capturar y descargar tus cookies en formato Netscape compatible directamente a un archivo de texto de forma totalmente privada y local.

### 🚀 Mejoras de este Fork:
* **Diseño Híbrido Real:** Interfaz responsiva corregida que se adapta a pantallas de PC y dispositivos móviles táctiles.
* **Corrección en Android:** Soluciona el error crítico de descargas bloqueadas en el entorno móvil de Firefox.
* **Modo Oscuro Nativo:** Paleta de colores optimizada para entornos de baja iluminación.
* **Privacidad:** Procesamiento 100% local. Tu información nunca sale de tu dispositivo.

### 🛠️ Instalación para Desarrollo:
1. Clona este repositorio.
2. Abre Firefox y navega a `about:debugging`.
3. Haz clic en "Este Firefox" (This Firefox) y selecciona "Cargar complemento temporal" (Load Temporary Add-on).
4. Selecciona el archivo `manifest.json` de la carpeta del proyecto.
